# Jogos Infantis Plínio Paganini 2025 – Boletim nº 6

Este repositório disponibiliza o Boletim nº 6 dos Jogos Infantis Plínio Paganini 2025, publicado pela Secretaria Municipal de Educação de Botucatu.

O documento apresenta:

- Classificação geral das escolas participantes;
- Resultados oficiais das modalidades esportivas;
- Chaveamentos das competições;
- Critérios de pontuação;
- Agenda oficial dos Jogos.

Na classificação geral, a EMEFI Elza Judith Carmelo Torres terminou em primeiro lugar com 108 pontos, conquistando o tricampeonato geral consecutivo.
